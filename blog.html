Question-1: What are the different ways to select an element in the DOM?

In JavaScript, there are several ways to select an element in the DOM (Document Object Model). Here are the most commonly used methods: getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll(), getElementsByName(), parentNode and children, closest(), firstChild, lastChild, nextSibling, previousSibling .

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML, innerText, and textContent are all properties used to get or set the content of HTML elements, but they behave differently. innerHTML: Returns or sets the HTML content inside an element, including tags. It can modify elements within the target. innerText: Returns or sets the visible text inside an element, excluding HTML tags and ignoring hidden content like display: none. textContent: Returns or sets the text content, including hidden elements, but ignores HTML tags, similar to innerText but without rendering concerns like CSS visibility.

Question-3: What is event delegation in the DOM?

Event delegation in the DOM is a technique where you attach a single event listener to a parent element instead of individual child elements. This is useful when you have many child elements that need to respond to the same event, such as click, mouseover, etc. When an event is triggered on a child element, it bubbles up to the parent element. The parent element, having the event listener attached, can handle the event for any of its child elements, even if they are added dynamically after the event listener is set. This improves performance and reduces memory usage.

Question-4: What is event bubbling in the DOM?

Event bubbling in the DOM is the process by which an event triggered on an element propagates (or "bubbles") up through its ancestor elements in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript, we can create, add, and remove elements from the DOM using the following methods: Creating an element: Use document.createElement() to create a new element. Adding an element: To add an element to the DOM, use methods like appendChild(), insertBefore(), or append()Removing an element: To remove an element, use removeChild() or remove():These methods allow you to dynamically manipulate the DOM by creating new elements, inserting them into the document, and removing them as needed.
